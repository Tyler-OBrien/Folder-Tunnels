Folder Tunnels

Quick Explanation: This is intended to expose a folder with a built website to requests on a randomly generated UUID path.

Requests with no path (/) are redirected to /index.html.

If you don't have any website files, download and extract the sample, and select that directory.

If you are using Firefox, you can change the live files and it will serve the latest version.
